# Features and Runtime

AristoByte is primarily a theme extension with a lightweight runtime layer for better onboarding.

## Theme assets

- `AristoByte Dark`
- `AristoByte Midnight`
- `AristoByte Dusk`
- `AristoByte High Contrast Dark`
- `AristoByte OLED`
- `AristoByte Light`
- `AristoByte Soft Light`
- `AristoByte High Contrast Light`

## Compatibility

- Supported VS Code version range: `>=1.90.0`

## Built-in dashboard experience

AristoByte includes a walkthrough-based dashboard for:

- Overview
- Installation
- Customization
- Accessibility
- Troubleshooting

## Commands

- `AristoByte: Open Home`
- `AristoByte: Open Getting Started Dashboard`
- `AristoByte: Open Documentation Index`
- `AristoByte: Open Marketplace/Homepage`
- `AristoByte: Apply Dark Theme`
- `AristoByte: Apply Light Theme`
- `AristoByte: Apply Midnight Theme`
- `AristoByte: Apply Dusk Theme`
- `AristoByte: Apply High Contrast Dark Theme`
- `AristoByte: Apply High Contrast Light Theme`
- `AristoByte: Apply OLED Theme`
- `AristoByte: Apply Soft Light Theme`

## Settings

- `aristobyteTheme.postInstallExperience`
- `aristobyteTheme.showWalkthroughOnInstallOrUpdate`
- `aristobyteTheme.defaultThemeOnInstall`
- `aristobyteTheme.openDocsInEditor`

## Activation behavior

- Activates on startup finish and AristoByte commands.
- Stores last-seen extension version.
- Optionally opens walkthrough after install/update.
- Optionally applies preferred default theme on first install.

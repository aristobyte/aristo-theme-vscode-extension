# Privacy

AristoByte Color Theme is a static theme extension.

## Data collection

- The extension does not implement custom telemetry.
- The extension does not send code or user content to external services.

## Third-party services

Distribution platforms (such as VS Code Marketplace, GitHub, or Open VSX) may process usage data according to their own policies.

## Security reporting

See `.github/SECURITY.md` for vulnerability disclosure steps.

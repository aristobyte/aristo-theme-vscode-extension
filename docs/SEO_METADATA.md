# Marketplace Metadata Checklist

This extension is distributed through VS Code Marketplace and Open VSX, not traditional web SEO pages.

## Metadata optimized in `package.json`

- `displayName`
- `description`
- `keywords`
- `categories`
- `homepage`
- `repository`
- `bugs`
- `icon`
- `galleryBanner`

## Metadata to keep updated for discoverability

- Screenshots in README
- Concise changelog entries
- Accurate versioning
- Theme labels and descriptions

# Support

## Community support

- GitHub Issues: bug reports and feature requests
- GitHub Discussions: usage questions and feedback

## What to include in reports

- OS and version
- VS Code version
- AristoByte extension version
- Selected theme variant
- Reproduction steps
- Screenshots

## Priority guidance

- Accessibility regressions: high priority
- Syntax readability regressions: high priority
- Cosmetic preference requests: normal priority

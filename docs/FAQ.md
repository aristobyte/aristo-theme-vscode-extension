# FAQ

## Is this extension only for dark mode?

No. It includes both dark and light variants.

## Does it support semantic highlighting?

Yes. It is designed to work with semantic highlighting enabled.

## Can I override colors?

Yes. Use VS Code `workbench.colorCustomizations` and `editor.tokenColorCustomizations`.

## Does this extension collect telemetry?

No custom telemetry is implemented by this extension.

## Where do I report bugs?

Use the GitHub issues page listed in `package.json`.

## How are releases published?

Releases are automated from `master` through the `Release` GitHub Actions workflow.
